# CortexLuna

CortexLuna 是简易，轻量化的 LLM 框架。建设在 Cordis 之上，与 Langchain 有一些相近的 API。

在设计上，CortexLuna 借鉴了 Langchain 和 Vercel AI SDK 的设计思路，并且在实现上也有一些借鉴。
